Research

Active Work

Ongoing · UMN MnRI

3D Truck Parking Detection & Segmentation

Fine-tuning VGGT for zero-shot 3D detection across structured and unstructured truck-parking environments. Replacing the DINOv2 patch encoder with DINOv3 for end-to-end retraining, and integrating SAM2/SAM3 for point cloud segmentation. Post-processing pipeline incorporates scale correction for VGGT's image downsampling, survey-coordinate boundary cutting, and cluster segmentation for slot-level occupancy estimation. Synthetic training assets generated in NVIDIA Isaac Sim and Unity 3D via a Docker-based pipeline. Advancing toward embodied applications including motion planning, with findings targeted for publication.

VGGT DINOv3 SAM2 Point Clouds Isaac Sim 3D Perception
Ongoing · UMN MnRI

3D Reconstruction from Pre-Calibrated Fisheye Imagery

Developing a dense scene geometry reconstruction pipeline from pre-calibrated fisheye images and video footage for autonomous driving perception. Extending the pipeline with active inference for vehicle and pedestrian motion prediction, advancing toward physical AI systems that unify perception, prediction, and control under a single free-energy minimization framework.

Graduate Projects

Course & Hackathon Work

CSCI 5551 · Intelligent Robotics

Autonomous Cable Routing

AprilTag-based cable routing on the UFACTORY Lite 6 arm using a ZED stereo camera. Implemented a robot state machine with look-ahead motion history for reliable multi-clip sequencing.

EE 5271 · Robot Vision

3D Truck Parking Detection Using Foundation Models

Comparative evaluation of VGGT and Depth Anything v3 for zero-shot 3D truck detection. Camera calibration via the Tsai method; RANSAC, DBSCAN, NOH, and CSF segmentation applied to reconstructed point clouds.

ME 5286 · Robotics

Pick & Place with CNN-Based Tool Recognition

Full pick-and-place pipeline on a UR5e for tool sorting. VGG-based CNN for tool recognition integrated with motion planning via Python and the RoboDK API.

CSCI 5512 · AI II

Predictive Spatial Temperature Control

Evaluated MPC and RL controllers for predictive HVAC control in a 3D thermal simulation with Gaussian Process spatial inference. Identified MPC as most stable; exposed a reward-shaping failure in the RL baseline.

AIIS Hackathon · UMN

DisasterBridge — Emergency Response Coordination

Real-time emergency coordination tool with an LLM-driven welfare call dispatcher generating prioritized responder cards for individuals with disabilities. GIS-based vulnerability mapping planned as next phase.
